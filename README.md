# ToolHub Frontend

Calls the backend API only — no Supabase configuration needed.

## Setup

```bash
npm install
```

Create `.env`:
```env
VITE_API_URL=/api
```

## Run

Start the backend first (`../toolhub-be`), then:

```bash
npm run dev
```

See [SETUP.md](../toolhub-be/SETUP.md) for full instructions.
